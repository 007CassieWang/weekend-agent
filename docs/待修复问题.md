### 1. 工具函数需要风险分级和结构化错误返回

**文件**: `tools.py`

当前工具函数直接返回 Python 对象，无统一错误格式。接入真实 API（支付、预约）后风险较高。

**建议**：
```python
from enum import Enum

class RiskLevel(Enum):
    READ_ONLY = 1      # search_activities, check_route_time
    DRAFT = 2          # build_candidate_plans
    SIDE_EFFECT = 3    # book_activity, order_item
    FINANCIAL = 4      # 真实支付

@dataclass
class ToolResult:
    success: bool
    data: Any
    error: Optional[str] = None
    risk_level: RiskLevel = RiskLevel.READ_ONLY
    requires_approval: bool = False
```

**预估工作量**: ~100 行代码

### 2. `check_route_time` 距离计算不基于真实坐标

**文件**: `tools.py:191-193`

```python
distance_km = 5.0  # 硬编码默认值
```

当接入真实 POI 数据后，所有路线时间计算会错误。

**建议**: 使用 Haversine 公式基于坐标计算近似距离：
```python
from math import radians, sin, cos, sqrt, atan2

def haversine(lat1, lon1, lat2, lon2):
    R = 6371  # 地球半径 (km)
    dlat = radians(lat2 - lat1)
    dlon = radians(lon2 - lon1)
    a = sin(dlat/2)**2 + cos(radians(lat1)) * cos(radians(lat2)) * sin(dlon/2)**2
    return R * 2 * atan2(sqrt(a), sqrt(1-a))
```

**预估工作量**: ~30 行代码
